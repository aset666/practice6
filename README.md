# ML FastAPI Docker — Iris Species Predictor

End-to-end ML system: Random Forest model served via **FastAPI**, with a **Streamlit** frontend and **MLflow** experiment tracking.

---

## Project Structure

```
ml-fastapi-docker/
├── train.py          # Train model + log to MLflow
├── main.py           # FastAPI app
├── app.py            # Streamlit frontend
├── model.joblib      # Saved model (generated by train.py)
├── requirements.txt  # Dependencies
├── Dockerfile        # Container instructions
└── README.md
```

---

## Setup

```bash
python -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

---

## Run (3 terminals)

### Terminal 1 — MLflow tracking server
```bash
mlflow ui
```
Opens at **http://127.0.0.1:5000**

### Terminal 2 — Train model
```bash
python train.py
```
Trains the model, saves `model.joblib`, logs params/metrics/artifacts to MLflow, registers model.

### Terminal 3 — FastAPI
```bash
uvicorn main:app --reload
```
API at **http://127.0.0.1:8000** | Swagger: **http://127.0.0.1:8000/docs**

### Terminal 4 — Streamlit frontend
```bash
streamlit run app.py
```
Opens at **http://localhost:8501**

## Docker

```bash
docker build -t ml-fastapi-docker .
docker run -p 8000:8000 ml-fastapi-docker
```

## API Reference

| Method | Path       | Description              |
|--------|------------|--------------------------|
| GET    | `/`        | Health check             |
| POST   | `/predict` | Predict Iris species     |

**POST /predict body:**
```json
{"features": [5.1, 3.5, 1.4, 0.2]}
```

**Response:**
```json
{
  "predicted_class_id": 0,
  "predicted_class_name": "setosa",
  "probabilities": {"setosa": 1.0, "versicolor": 0.0, "virginica": 0.0}
}
```

## MLflow

- Experiment: `Iris-Classification`
- Logged: `n_estimators`, `max_depth`, `accuracy`, `f1_score`, model artifact
- Model Registry name: `IrisRandomForest`

View at **http://127.0.0.1:5000**
